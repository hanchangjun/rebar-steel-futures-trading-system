# Vibe Coding 平台集成配置查看指南

## 📋 概述

本指南将帮助您了解如何在 Vibe Coding 平台上查看和管理集成配置（包括企业微信机器人、大模型、数据库等）。

---

## 🔍 方法 1: 使用检查脚本（推荐）

我们已经为您创建了集成配置检查脚本，可以快速查看所有集成的配置状态。

### 运行检查脚本

```bash
cd /workspace/projects
python check_integration.py
```

### 检查内容

脚本会检查以下内容：

1. **环境变量**
   - `COZE_WORKLOAD_IDENTITY_API_KEY` - API 密钥
   - `COZE_INTEGRATION_MODEL_BASE_URL` - 模型服务地址
   - `COZE_WORKSPACE_PATH` - 工作目录路径

2. **企业微信集成**
   - 集成是否已配置
   - Webhook Key 是否有效
   - 配置类型（完整 URL 或仅 Key）

3. **大模型集成**
   - 模型服务是否已配置
   - Base URL 是否正确

4. **Agent 配置**
   - 配置文件是否存在
   - 使用的模型
   - 温度设置
   - 工具数量

### 测试企业微信集成

在检查完成后，脚本会询问是否测试企业微信集成。输入 `y` 或 `yes` 即可发送测试消息：

```
是否要测试企业微信集成？
  输入 'y' 或 'yes' 进行测试
  输入其他任何内容跳过

请选择: y
```

## 🧪 方法 4: 通过 API 查看

如果您有 API 访问权限，可以通过 API 查看集成配置：

### 示例代码

```python
from coze_workload_identity import Client
import json

# 创建客户端
client = Client()

# 获取企业微信集成配置
try:
    credential = client.get_integration_credential('integration-wechat-bot')
    print("企业微信集成配置:")
    print(json.dumps(json.loads(credential), indent=2, ensure_ascii=False))
except Exception as e:
    print(f"企业微信集成未配置: {e}")

# 获取其他集成配置
integrations = [
    'integration-wechat-bot',
    'integration-llm',
    'integration-oss',
]

for integration_id in integrations:
    try:
        credential = client.get_integration_credential(integration_id)
        print(f"\n{integration_id}: ✅ 已配置")
    except Exception as e:
        print(f"\n{integration_id}: ❌ 未配置 ({e})")
```

## 🔧 配置企业微信集成（如未配置）

如果企业微信集成未配置，请按照以下步骤配置：

### 步骤 1: 在企业微信群添加机器人

1. 打开企业微信群
2. 点击右上角 `...` → 选择 **"群机器人"**
3. 点击 **"添加机器人"**
4. 输入机器人名称（如：`螺纹钢期货助手`）
5. 复制显示的 Webhook URL

### 步骤 2: 在 Vibe Coding 平台配置

1. 登录 Vibe Coding 平台
2. 进入项目 → **集成** 或 **"Integrations"**
3. 搜索 **"企业微信机器人"** 或 **"WeChat Bot"**
4. 点击 **"添加集成"**
5. 粘贴 Webhook URL
6. 点击 **"保存"**

### 步骤 3: 测试配置

运行测试脚本验证配置：

```bash
cd /workspace/projects
python check_integration.py
```

输入 `y` 或 `yes` 进行测试。

## 💡 常见问题

### Q1: 为什么在平台上找不到"集成"入口？

**A**: 可能的原因：
- 您的账号没有集成管理权限
- 项目不支持集成功能
- 联系技术支持确认平台版本

### Q2: 集成配置在哪里修改？

**A**:
1. 进入集成管理页面
2. 找到目标集成
3. 点击 **"编辑"** 或 **"修改配置"**
4. 修改配置后保存

### Q3: 如何删除集成配置？

**A**:
1. 进入集成管理页面
2. 找到目标集成
3. 点击 **"删除"**
4. 确认删除操作

### Q4: 集成配置修改后需要重启应用吗？

**A**: 取决于集成类型：
- 企业微信机器人：不需要重启，即时生效
- 大模型：通常不需要重启
- 数据库：可能需要重启应用
