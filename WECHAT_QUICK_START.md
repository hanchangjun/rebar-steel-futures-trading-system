## ⚠️ 配置步骤

要使用企业微信通知功能，需要完成以下配置：

### 步骤 1: 在企业微信群添加机器人

1. 打开目标企业微信群
2. 点击右上角 "..."（更多）
3. 选择"群机器人" → "添加机器人"
4. 输入机器人名称（如：螺纹钢期货助手）
5. 点击"添加"
6. 复制 Webhook URL（格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=XXXXX`）

### 步骤 2: 配置系统集成

在 Vibe Coding 平台中：

1. 进入项目设置
2. 找到"集成"或"Integrations"
3. 添加"企业微信机器人"集成 (`integration-wechat-bot`)
4. 粘贴 Webhook URL
5. 保存配置

### 步骤 3: 验证配置

发送测试消息：
```
请帮我发送一条测试消息到企业微信
```

如果收到消息，说明配置成功！

## 🔍 故障排查

### 问题: 无法获取企业微信 webhook key

**原因**: 系统集成未配置或配置错误

**解决方案**:
1. 确认已在 Vibe Coding 平台添加企业微信集成
2. 检查 Webhook URL 是否正确
3. 重新配置集成信息

### 问题: 消息发送失败

**可能原因**:
- Webhook URL 错误
- 网络连接问题
- 企业微信群已解散或机器人被移除

**解决方案**:
1. 检查 Webhook URL 是否有效
2. 测试网络连接
3. 重新添加机器人
