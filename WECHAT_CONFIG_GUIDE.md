# 企业微信集成配置指南

## 📋 概述

本指南将帮助您完成企业微信机器人的配置，实现螺纹钢期货买卖点交易系统的自动通知功能。

---

## 🎯 配置目标

在完成本指南后，您将能够：
- ✅ 在企业微信群中添加机器人
- ✅ 在 Vibe Coding 平台配置企业微信集成
- ✅ 测试企业微信通知功能
- ✅ 接收市场分析和交易信号推送

---

## 📝 配置步骤

### 步骤 1: 在企业微信群添加机器人

#### 1.1 打开企业微信群

1. 打开企业微信客户端（PC 端或手机端）
2. 进入需要接收通知的目标群聊

#### 1.2 添加群机器人

**PC 端操作步骤：**
1. 点击群聊右上角的 `...`（更多）按钮
2. 选择 **"群机器人"**
3. 点击 **"添加机器人"**
4. 输入机器人名称（如：`螺纹钢期货助手`）
5. 点击 **"添加"**

**手机端操作步骤：**
1. 点击群聊右上角的 `...`（更多）按钮
2. 找到 **"群机器人"**
3. 点击 **"添加机器人"**
4. 输入机器人名称（如：`螺纹钢期货助手`）
5. 点击 **"添加"**

#### 1.3 获取 Webhook URL

添加成功后，系统会显示 Webhook URL，格式如下：

```
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=XXXXX-XXXXX-XXXXX-XXXXX
```

**重要提示：**
- ⚠️ **请妥善保管此 URL**，不要泄露给无关人员
- 📋 **复制此 URL**，后续配置需要使用
- 🔒 如果机器人被删除或 URL 泄露，可以重新添加获取新的 URL

---

### 步骤 2: 在 Vibe Coding 平台配置集成

#### 2.1 登录 Vibe Coding 平台

1. 打开浏览器，访问 Vibe Coding 平台
2. 登录您的账号
3. 进入项目页面

#### 2.2 添加企业微信集成

1. 在项目页面，找到 **"集成"** 或 **"Integrations"** 选项
2. 点击进入集成管理页面
3. 搜索 **"企业微信机器人"** 或 **"WeChat Bot"**
4. 点击 **"添加集成"** 或 **"配置集成"**

#### 2.3 配置集成参数

在集成配置页面，填写以下信息：

| 参数名称 | 填写内容 | 说明 |
|---------|---------|------|
| **集成名称** | `企业微信机器人` | 自定义名称 |
| **集成 ID** | `integration-wechat-bot` | 系统自动生成 |
| **Webhook URL** | 步骤 1.3 中复制的完整 URL | 必填 |

**示例：**

```
集成名称: 企业微信机器人
Webhook URL: https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=0e8b6efe-5284-49da-807d-a997b7553c16
```

#### 2.4 保存配置

1. 确认配置信息无误
2. 点击 **"保存"** 或 **"确认"**
3. 等待系统提示配置成功

---

### 步骤 3: 验证配置

#### 3.1 发送测试消息

在系统的对话界面中，输入以下测试命令：

```
请帮我发送一条测试消息到企业微信，内容是：企业微信通知配置测试成功！
```

#### 3.2 检查企业微信群

1. 打开企业微信群
2. 查看是否收到测试消息
3. 消息内容应该为：`企业微信通知配置测试成功！`

#### 3.3 确认配置成功

如果收到测试消息，说明配置成功！🎉

如果未收到消息，请参考 [故障排查](#故障排查) 部分。

## 🔍 故障排查

### 问题 1: 找不到"群机器人"选项

**可能原因：**
- 您可能没有添加机器人的权限
- 群聊可能禁用了机器人功能

**解决方案：**
1. 联系群主或管理员，确认是否允许添加机器人
2. 确认您是否有权限管理群机器人

---

### 问题 2: Vibe Coding 平台找不到"集成"选项

**可能原因：**
- 账号权限不足
- 平台版本不支持

**解决方案：**
1. 联系技术支持，确认账号权限
2. 确认平台版本是否支持集成功能

---

### 问题 3: 测试消息发送失败

**可能原因：**
- Webhook URL 配置错误
- 企业微信机器人被删除
- 网络连接问题
- 集成配置未生效

**解决方案：**

#### 3.1 检查 Webhook URL

1. 确认 Webhook URL 格式是否正确
2. 确认 URL 中的 `key=` 后面有正确的密钥
3. 重新复制并配置 Webhook URL

#### 3.2 检查企业微信机器人

1. 进入企业微信群
2. 查看机器人是否还在群中
3. 如果机器人被删除，重新添加并更新 Webhook URL

#### 3.3 检查网络连接

1. 确认网络连接正常
2. 确认 Vibe Coding 平台可以访问外部网络
3. 尝试使用浏览器访问 Webhook URL（会返回错误，但可以测试连通性）

#### 3.4 检查集成配置

1. 登录 Vibe Coding 平台
2. 进入集成管理页面
3. 确认企业微信集成已正确配置
4. 重新保存集成配置

---

### 问题 4: 发送消息后企业微信群无反应

**可能原因：**
- 消息内容包含敏感词
- 消息格式不正确
- 机器人被禁言

**解决方案：**

1. **检查消息内容**
   - 确认消息不包含敏感词
   - 尝试发送简单的文本消息

2. **检查消息格式**
   - Markdown 消息最长 4096 字节
   - 文本消息最长 2048 字节
   - 确认消息长度未超限

3. **检查机器人状态**
   - 联系群主确认机器人是否被禁言
   - 确认机器人权限设置

## 💡 配置最佳实践

### 1. 安全性建议

- 🔒 不要在公开场所分享 Webhook URL
- 🔑 定期更换 Webhook URL（建议每 3 个月）
- 🚫 不要将 Webhook URL 提交到代码仓库
- 📝 记录机器人添加日期，便于管理

### 2. 稳定性建议

- 📱 建议为重要的通知功能配置备用群
- 🔄 定期测试机器人是否正常工作
- 📊 监控消息发送成功率
- ⚠️ 设置异常告警，及时发现问题

### 3. 使用建议

- 🎯 合理设置消息推送频率，避免骚扰群成员
- 📝 使用统一的消息格式，提高可读性
- 🕐 在交易时间内推送重要信息
- 📊 使用 Markdown 格式，美化消息展示
