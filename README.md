# 螺纹钢期货买卖点交易系统

## 📖 系统概述

本系统是一个基于 AI 智能体的螺纹钢期货交易分析系统，集成了实时行情获取、技术指标计算、买卖点识别、风险提示等核心功能。系统能够帮助用户快速分析市场状况，识别交易机会，并提供专业的交易建议。

### 核心特性

- 📊 **实时行情查询**：获取螺纹钢期货主力合约的实时报价
- 📈 **技术指标分析**：支持 MA、EMA、MACD、RSI、KDJ、布林线等 8 种技术指标
- 🎯 **买卖点识别**：基于多指标综合判断，识别买入/卖出/观望信号
- 📰 **市场资讯**：获取最新市场新闻和专业分析报告
- ⚠️ **风险提示**：提供止损、止盈建议和风险控制建议
- 🤖 **智能对话**：支持自然语言交互，易于使用

## 🛠️ 功能模块

### 1. 数据获取工具

#### 1.1 实时行情查询
- **工具名称**：`get_futures_realtime_quotes`
- **功能**：获取螺纹钢期货主力合约的实时行情数据
- **参数**：
  - `symbol`：期货品种名称（默认：螺纹钢）
  - `contract`：合约代码（可选，如 RB2505）
- **返回**：当前价格、涨跌幅、成交量等实时数据

#### 1.2 历史数据查询
- **工具名称**：`get_futures_historical_data`
- **功能**：查询历史K线数据
- **参数**：
  - `symbol`：期货品种名称
  - `contract`：合约代码
  - `period`：K线周期（日K、60分钟、15分钟等）
  - `days`：历史天数
- **返回**：历史K线数据来源链接

#### 1.3 市场新闻查询
- **工具名称**：`get_futures_market_news`
- **功能**：获取最新的市场新闻和资讯
- **参数**：
  - `symbol`：期货品种名称
  - `count`：获取新闻数量
- **返回**：新闻标题、来源、链接、发布时间

#### 1.4 分析报告查询
- **工具名称**：`get_futures_analysis_report`
- **功能**：获取专业机构的分析报告和研报
- **参数**：
  - `symbol`：期货品种名称
- **返回**：研报标题、摘要、来源

#### 1.5 综合市场信息
- **工具名称**：`get_comprehensive_market_info`
- **功能**：一键获取行情、新闻、分析的综合信息
- **参数**：
  - `symbol`：期货品种名称
  - `contract`：合约代码（可选）
- **返回**：包含行情、新闻、分析的完整信息

### 2. 技术指标计算工具

#### 2.1 移动平均线 (MA)
- **工具名称**：`calculate_ma`
- **功能**：计算简单移动平均线
- **参数**：
  - `data`：K线数据列表（包含 close 字段）
  - `period`：均线周期（默认：20）
- **用途**：判断趋势方向，识别支撑和压力位

#### 2.2 指数移动平均线 (EMA)
- **工具名称**：`calculate_ema`
- **功能**：计算指数移动平均线
- **参数**：
  - `data`：K线数据列表
  - `period`：均线周期（默认：12）
- **用途**：比 MA 更敏感的趋势指标

#### 2.3 MACD 指标
- **工具名称**：`calculate_macd`
- **功能**：计算 MACD 指标
- **参数**：
  - `data`：K线数据列表
  - `fast_period`：快线周期（默认：12）
  - `slow_period`：慢线周期（默认：26）
  - `signal_period`：信号线周期（默认：9）
- **用途**：识别趋势和买卖信号（金叉/死叉）

#### 2.4 RSI 指标
- **工具名称**：`calculate_rsi`
- **功能**：计算相对强弱指标
- **参数**：
  - `data`：K线数据列表
  - `period`：RSI 周期（默认：14）
- **用途**：判断超买超卖（>70 超买，<30 超卖）

#### 2.5 KDJ 指标
- **工具名称**：`calculate_kdj`
- **功能**：计算随机指标
- **参数**：
  - `data`：K线数据列表（包含 high、low、close）
  - `n`：周期（默认：9）
  - `m1`：K 值平滑系数（默认：3）
  - `m2`：D 值平滑系数（默认：3）
- **用途**：识别短期买卖信号

#### 2.6 布林线 (Bollinger Bands)
- **工具名称**：`calculate_bollinger_bands`
- **功能**：计算布林线指标
- **参数**：
  - `data`：K线数据列表
  - `period`：周期（默认：20）
  - `std_dev`：标准差倍数（默认：2.0）
- **用途**：判断价格波动区间和突破信号

#### 2.7 成交量分析
- **工具名称**：`analyze_volume`
- **功能**：分析成交量变化
- **参数**：
  - `data`：K线数据列表（包含 volume）
  - `period`：平均成交量周期（默认：5）
- **用途**：判断市场活跃度和动能

#### 2.8 综合指标分析
- **工具名称**：`calculate_all_indicators`
- **功能**：一次性计算所有技术指标
- **参数**：
  - `data`：K线数据列表（包含 open、high、low、close、volume）
- **返回**：所有指标的综合分析结果

## 🎯 买卖点识别标准

### 买入信号（满足多个条件时强度更高）

1. **趋势信号**
   - 价格站上短期均线（如5日、10日线）
   - 短期均线上穿长期均线（金叉）
   - MACD金叉（DIF上穿DEA）
   - KDJ金叉（K上穿D，且K<50）

2. **超卖信号**
   - RSI < 30（超卖区域）
   - 价格触及布林线下轨
   - KDJ的K值和D值均 < 20

3. **背离信号**
   - 价格创新低但指标不创新低（底背离）

4. **支撑信号**
   - 价格回踩重要支撑位后企稳
   - 在支撑位出现止跌信号

5. **成交量信号**
   - 上涨时成交量放大
   - 下跌时成交量萎缩

### 卖出信号

1. **趋势信号**
   - 价格跌破短期均线
   - 短期均线下穿长期均线（死叉）
   - MACD死叉（DIF下穿DEA）
   - KDJ死叉（K下穿D，且K>50）

2. **超买信号**
   - RSI > 70（超买区域）
   - 价格触及布林线上轨
   - KDJ的K值和D值均 > 80

3. **背离信号**
   - 价格创新高但指标不创新高（顶背离）

4. **压力信号**
   - 价格接近重要压力位
   - 在压力位遇阻回落

5. **成交量信号**
   - 上涨时成交量萎缩
   - 下跌时成交量放大
